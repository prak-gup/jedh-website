## ðŸš€ How to Use the Website

### 1. **Local Testing**
```bash
# Open any HTML file in browser
open website/index.html

# Or use a local server
cd website
python -m http.server 8000
# Visit http://localhost:8000
```

### 2. **Updating Content**

**Add New Doctor**:
1. Open `doctors.html`
2. Copy existing doctor card
3. Update name, specialty, qualifications
4. Update experience and stats
5. Change data-specialty="eye" or "dental"

**Update Pricing**:
1. Open `pricing.html`
2. Find the pricing table
3. Update rows with new prices
4. Ensure consistency with service pages

**Add Blog Post**:
1. Open `blog/index.html`
2. Copy existing blog-card
3. Update category, title, content
4. Change author and read time
5. Update data-category attribute

### 3. **Customization**

**Change Colors**:
```css
/* In style.css */
:root {
    --primary-color: #0066CC;  /* Change this */
    --secondary-color: #00A86B; /* And this */
}
```

**Update Contact Info**:
- Search for `+91 9602227267` in all HTML files
- Search for `dramit.eye@gmail.com`
- Replace with new info

**Add Insurance Logos**:
1. Place logos in `assets/images/Cashless/`
2. Update `loadInsuranceLogos()` in `main.js`

### 4. **Deploy to Production**

**Option A: Static Hosting (Netlify, Vercel)**
```bash
# Deploy entire website folder
netlify deploy --dir=website --prod
```

**Option B: WordPress Integration**
- Use existing WordPress site
- Replace pages with these HTML versions
- Or use as template for WordPress theme

**Option C: Traditional Hosting**
```bash
# Upload via FTP
# Point to website/ as root directory
# Update domain in all canonical URLs
```
